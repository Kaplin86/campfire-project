# Cat in the Cavern
## A project for Hackclub's Campfire hackathon
Cat in the cavern is a platformer-esque game where you must go from the bottom of a procedurally generated cave to the top, running into many enemies and difficult jumps.
You also need to make sure the cat that follows you stay alive.

Also was used as a learning experience for the team.

## Controls
Mouse-click to attack, shift mouse click to place blocks

## Status
Playable

## Development / Packaging
- Download Godot 4.6.1
- Load the game using the import button
- That's all!
